Call SimpleBox

$.simpleBox();

Markup

By default, simplebox targets container with class of .simpleBox and works with atleast the specified markup below. A container, a selector li, and an image with a data-attribute of data-sbox-img.

<ul class="simpleBox">
    <li>
        <img src="imageThumb.jpg" data-sbox-img="imageFull.jpg" alt="" />
    </li>
</ul>

Styles

Although this demo uses SimpleGrid for flush grid systems - the only required styles are listed below.

*,
*:after,
*:before {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

ul {
    list-style: none;
}

li, 
.filler {
    display: inline-block;
    position: relative;
    width: calc(100%/6); // number of columns
    -webkit-transform: translate3d(0,0,0); // hover transition fix for webkit
}

li > *:not(img) {
    display: none;
}

li img {
    max-width: 100%;
    width: 100%;
    display: none;
}

.simpleBox-show {   
    position: relative;
    width: 100%;
    float: left;
    display: none;
}

.simpleBox-current-img {
    max-width: 100%;
}

li .overlay {
    
}

.simpleBox-content {
    
}

.simpleBox-close {

}

// Responsivenesssssssss
@media only screen and (max-width: 960px) {
    li,
    .filler {
        width: calc(100%/5);
    }
    .simpleBox-content {
        display: block;
    }
}

@media only screen and (max-width: 768px) {
    li,
    .filler {
        width: calc(100%/4);
    }
}

@media only screen and (max-width: 640px) {
    li, 
    .filler {
        width: calc(100%/3);
    }
}

@media only screen and (max-width: 480px) {
    li,
    .filler {
        width: calc(100%/2);
    }
}

Advanced

However, it can work with any markup and any container class by setting a container and options. If selector in options is not li, then the styles should be updated to reflect selector.

$('.container').simpleBox({
    duration: 350,          // vary the speed of animation
    easing: 'swing',        // the easing function for animation // expo, cubic, circ, swing
    activeClass: 'active',  // class given to the active element
    selector: 'li',         // defaults to li
    sequentialLoad: true,   // fade in images sequentially
    overlay: true           // hover overlay
});
